cd nomeDiretorio (para acessar o diretório)
cd.. (para voltar ao diretório anterior)
dir (para vir todos os diretórios)
git clone URLdoRepositório (para clonar um projeto)
git log (para ver todos os comites de um repositório detalhadamente)
git log --oneline (para ver todos os comites de forma resumida)
git pull URLdorepositorio (para ver atualizações no projeto)
git add nomeDiretório (para add no próximo comando que é o commit)
git commit nomeDiretorio -m "Aqui vai uma mensagem dizendo quais foram as atualizações no arquivo" (para fazer o commit)
git push origin main (para enviar minhas atualizações para a origem)
git restore --source hash nomeDiretorio (voltar na versão de um commit anterior)
git checkout -b nomeBranch (criar e acessar uma nova branch)
git switch nomeBranch (trocar de branch)
git branch (ver quais branchs existentes e em qual estou)
git merge nomeBranch (primeiro entrar na branch main e depois dar este comando para fazer o merge)
